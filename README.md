# JTLH LTD

Landing site for **JTLH LTD** — AI consultation and integration.

## Stack

- Next.js + TypeScript
- Tailwind CSS
- Resend (contact form)
- Render web service

## Local development

```bash
npm install
cp .env.example .env.local
# add RESEND_API_KEY
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Contact

Form posts to `/api/contact` and emails **jtlhltd@gmail.com** via Resend.
