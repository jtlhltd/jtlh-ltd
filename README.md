# JTLH LTD

Landing site for **JTLH LTD** — AI consultation and integration.

## Stack

- Next.js (static export)
- TypeScript
- Tailwind CSS
- Deployed on Render

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Static output lands in `out/`.
